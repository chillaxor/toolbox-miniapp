# PRD · 微信小程序工具箱

> 版本：v1.0 | 日期：2026-05-29 | 产品经理：许清楚

---

## 1. 产品目标

打造一款**日常高频、开箱即用**的微信小程序工具合集，让用户在一个入口内快速完成生活计算、日期时间、文本处理、图片处理四大类日常任务，无需安装多个 App。

---

## 2. 用户故事

| # | 故事 |
|---|------|
| US-1 | 作为**普通用户**，我想快速算出自己的 BMI 并查看健康建议，以便了解自身健康状态。 |
| US-2 | 作为**职场人士**，我想在微信里直接格式化粘贴来的 JSON 数据，以便快速排查接口问题，不需要切换到电脑。 |
| US-3 | 作为**出行用户**，我想随时查询实时汇率并换算金额，以便出境旅游时做消费决策。 |
| US-4 | 作为**项目经理**，我想计算两个日期之间的工作日天数，以便合理安排项目排期。 |
| US-5 | 作为**内容创作者**，我想在手机上压缩图片或生成二维码，以便减少发布素材的体积并分享链接。 |

---

## 3. 功能需求池

### P0 · 核心必有（首版上线）

**生活计算**
- [ ] BMI 计算（输入身高体重，输出 BMI 值 + 健康等级说明）
- [ ] 单位换算（长度、重量、温度三类）
- [ ] 汇率转换（内置常用币种，支持手动刷新汇率）
- [ ] 税后工资计算（五险一金比例可配置，输出实发工资）

**日期时间**
- [ ] 万年历（月视图，支持翻月，标注节假日与农历）
- [ ] 倒计时（设定目标日期，实时显示剩余天/时/分/秒）
- [ ] 工作日计算（选择起止日期，自动扣除周末与法定节假日）
- [ ] 农历转换（公历↔农历互转，显示生肖、节气信息）

**文本处理**
- [ ] 字数统计（字符数、汉字数、单词数、行数）
- [ ] 大小写转换（全大写、全小写、首字母大写）
- [ ] JSON 格式化（美化 / 压缩，错误高亮提示）
- [ ] Base64 编解码（文本 → Base64 / Base64 → 文本）

**图片工具**
- [ ] 图片压缩（选取本地图片，设置质量比例，输出压缩后图片）
- [ ] 图片转 Base64（选取图片，输出 Base64 字符串并支持复制）
- [ ] 二维码生成（输入任意文本/URL，生成可保存的二维码图片）

**基础框架**
- [ ] 首页工具导航（分类彩色卡片网格，4 大分类 + 16 个工具入口）
- [ ] 底部 Tab 导航（首页 / 收藏 / 历史 / 我的）

---

### P1 · 重要功能（首版后快速迭代）

- [ ] **工具搜索**：首页顶部搜索框，支持工具名称模糊搜索
- [ ] **收藏工具**：长按或工具内 ❤ 按钮收藏，首页收藏区快捷入口
- [ ] **历史记录**：自动保存每个工具最近 10 条使用记录（本地存储）
- [ ] **深色模式**：跟随系统设置自动切换
- [ ] **单位换算扩展**：增加面积、体积、速度分类

---

### P2 · 锦上添花（后续版本）

- [ ] **桌面快捷方式**：支持将常用工具添加到桌面（小程序快捷方式）
- [ ] **分享工具结果**：生成结果卡片图片分享到微信好友/朋友圈
- [ ] **汇率自动刷新**：后台定时拉取最新汇率（需接入汇率 API）
- [ ] **节假日数据更新**：每年初更新法定节假日数据包
- [ ] **工具使用统计**：「我的」页展示使用次数最多的工具 Top5

## 5. UI 设计规范

### 5.1 分类主题色

| 分类 | 主题色 | Hex | 辅助色 |
|------|--------|-----|--------|
| 生活计算 | 珊瑚橙 | `#FF6B35` | `#FFE5D9` |
| 日期时间 | 天空蓝 | `#4ECDC4` | `#D8F5F3` |
| 文本处理 | 草绿色 | `#45B058` | `#D9F0DC` |
| 图片工具 | 薰衣草紫 | `#9B59B6` | `#EFD9F7` |

### 5.2 全局色彩

```
背景色（浅）：#F8F9FA
背景色（深色模式）：#1A1A2E
卡片背景：#FFFFFF
文字主色：#2C3E50
文字次色：#7F8C8D
分割线：#ECEFF1
成功色：#27AE60
警告色：#F39C12
错误色：#E74C3C
```

### 5.3 组件规范

**工具卡片（首页网格）**
- 尺寸：`calc(50% - 12px)` 宽，高度 `100rpx`（横向双列）
- 圆角：`16rpx`
- 内容：左侧工具图标（40rpx）+ 右侧工具名称（28rpx 字号）
- 背景：对应分类辅助色（浅色底），左侧色条为主题色
- 阴影：`0 4rpx 12rpx rgba(0,0,0,0.08)`
- 按压态：`scale(0.97)` + 加深背景色

**主操作按钮**
- 宽度：`100%`，高度：`88rpx`
- 圆角：`44rpx`（胶囊形）
- 背景：对应分类主题色
- 字色：白色，字号：`32rpx`，字重：`bold`

**输入框**
- 高度：`88rpx`，圆角：`12rpx`
- 边框：`2rpx solid #ECEFF1`（聚焦时变主题色）
- 内边距：`0 24rpx`

**分类标题**
- 左侧 `8rpx` 彩色竖条（主题色）+ 文字
- 字号：`30rpx`，字重：`600`

### 5.4 图标规范
- 使用 iconfont 或 Lucide 风格线性图标
- 首页卡片图标尺寸：`40rpx × 40rpx`
- 颜色：对应分类主题色

| # | 问题 | 影响范围 | 优先级 |
|---|------|----------|--------|
| Q1 | 汇率数据来源：使用免费开放 API（如 exchangerate.host）还是内置静态数据？静态数据更新频率如何？ | 汇率转换工具 | 高 |

| Q5 | 首版是否需要用户体系（登录/账号）？收藏与历史记录是否只做本地存储即可？ | 收藏、历史、我的 | 高 |
